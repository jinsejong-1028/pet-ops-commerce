# 프로젝트 개요

## 프로젝트명

PetOps Commerce

## 한 줄 설명

반려동물 상품 커머스의 주문, 재고, 쿠폰, 운영 데이터를 관리하는 Java/Spring Boot 기반 백엔드 플랫폼입니다.

## 배경

최근 백엔드 채용 공고에서는 단순 API 구현보다 아래 역량을 함께 요구하는 흐름이 강합니다.

- Java/Spring Boot/JPA 기반 서비스 개발
- RESTful API 설계
- RDBMS 모델링과 성능 개선
- Redis, Message Queue 기반 분산 처리
- Docker, AWS, CI/CD 기반 운영 이해
- 모니터링, 로그, 장애 대응 경험
- AI/LLM 또는 데이터 기반 기능 개선 경험

이 프로젝트는 위 요구사항을 하나의 완성 가능한 서비스 안에서 보여주는 것을 목표로 합니다.

## 포트폴리오 메시지

PHP/MySQL 실무 경험을 바탕으로 익숙한 업무 도메인인 주문, 재고, 운영 관리를 Java/Spring 생태계로 재설계했습니다. 단순히 새로운 문법을 학습한 것이 아니라, 운영 가능한 백엔드 시스템의 구조, 데이터 모델, 성능, 장애 대응, 배포 구조까지 고려했다는 점을 보여줍니다.

## 핵심 도메인

- 회원과 권한
- 반려동물 프로필
- 상품과 카테고리
- 장바구니
- 주문과 결제 상태
- 쿠폰과 프로모션
- 창고와 재고
- 재고 입출고 이력
- 운영 이벤트 로그
- 배치 통계
- AI 기반 운영 보조 기능

## 최종 결과물

- Spring Boot 백엔드 API 서버
- Docker Compose 기반 로컬 운영 환경
- Nginx 로드밸런싱 구성
- Redis 캐시/분산락 적용
- Message Queue 기반 비동기 처리
- Batch 기반 일별 통계/정산
- Swagger API 문서
- 테스트 코드와 Testcontainers
- 성능 개선/트러블슈팅 문서
